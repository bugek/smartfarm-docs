# Product Vision

SmartFarm is a software platform for agricultural operations that starts with **GAP readiness** and grows into an advisory and data platform.

We are GAP-first. Every Phase 1 feature must serve the chain:

> capture work in the field → attach evidence → make a record an expert can review → produce an audit-ready trail.

## Phase 1 outcome

A working GAP operating system for one compliance scheme, used end-to-end by:

- **farmers** recording work and attaching evidence in the field
- **experts** reviewing readiness and flagging missing evidence
- **compliance leads** preparing audit-ready records for an organization
- **admins** managing organization, members, farms, and crop cycles

Phase 1 is "done" when an organization can take a single crop cycle from setup through expert sign-off with traceable records and evidence.

## Out of scope for Phase 1 (short note only)

The following are deliberately deferred. They are mentioned only so engineers know not to design Phase 1 around them:

- automated agronomic recommendations (fertilizer, crop protection)
- AI-assisted case review (will reuse the structured records produced now)
- expert reputation marketplace and commerce
- model training pipelines

Phase 1 must keep the data model clean enough that these can be added later without rework, but must not build any of them now.
